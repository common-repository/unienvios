Tenha os métodos de envio da Unienvios na sua loja!

== Description ==

O plug-in unienvios tem como principal função integrar o woocommerce com a API unienvios.

== Installation ==

* Envie os arquivos do plugin para a pasta wp-content/plugins, ou instale usando o instalador de plugins do WordPress.
* Ative o plugin.
* Acesse no menu WooCommerce -> Configurações -> Entrega e clique em "Salvar" em cada um dos métodos de envio da Unienvios disponíveis.


= Requerimentos: = 

É necessário possuir uma conta no [Unienvios](https://unienvios.com.br) e ter instalado o [WooCommerce](http://wordpress.org/plugins/woocommerce/).

= Configurações do Plugin: =

Com o plugin instalado acesse o admin do WordPress e entre em "Unienvios"  e preencha as opções de login que você pode encontrar dentro da sua conta da Unienvios.

Depois acesse no menu WooCommerce -> Configurações -> Entrega e clique em salvar em cada um dos métodos de envio da Unienvios(Atenção isso é importante para que os métodos aparareçam no carrinho).

Pronto, sua loja já pode receber os métodos de envio da Unienvios.

Mais informações sobre as configurações do plugin em: [https://unienvios.com.br](https://unienvios.com.br).

== Frequently Asked Questions ==

= Como faço para que os métodos de envio apareçam em minha loja? =

Basta registrar seu e-mail e senha de API unienvios na página do plug-in
